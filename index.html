ISPASS 2024 Poster

Accel-Bench: Exploring the Potential of Programming using Hardware-Accelerated Functions
May 6, 2024
Presentation at the 2024 IEEE International Symposium on Performance Analysis of Systems and Software (ISPASS). This project focuses on a benchmark suite that aims to capture the performance of accelerator-intensive programming. Poster can be found here.

IWOCL 2022 Poster

IWOCL 2022 Poster: Embedding a DSL in SYCL for Productive and Performant Tensor Computing on Heterogeneous Devices
May 11, 2022
Presentation at the 2022 10th International Workshop on OpenCL (IWOCL) and SYCL. This project focuses on combining data parallelism provided though the combination of Data Parallel C++ (DPC++), and Temporal To Spatial Programming (T2SP) to provide a hardware agnostic programming model to construct new tensor computations. Presentation slides can be found here.
